Saviez-vous que vous pouvez cliquer sur une image pour l'utiliser sans la télécharger ?

Pour terminer cette tâche :

  • Cliquez sur l'image pour ouvrir la bibliothèque d'images (alternez l'affichage avec l'icône de la barre d'outils ou en double-cliquant).
  • Dans la bibliothèque d'images : survolez les miniatures pour les prévisualiser
  • Cliquez sur une miniature pour la placer sur votre page, ou cliquez sur l'icône pour la télécharger d'abord

Vous pouvez faire glisser des éléments n'importe où sur la page pour les réorganiser.

Pour terminer cette tâche :

  1. Cliquez et maintenez le clic sur une note pour commencer à la faire glisser.
